Prologue

以下Numberal System中最基本的符号：
$$
\begin{array}{l}
\mathbb{N}=\text { the set of positive integers (not including zero) } \
\mathbb{Z}=\text { the set of integers } \
\mathbb{Q}=\text { the set of rational numbers } \
\mathbb{R}=\text { the set of real numbers } \
\mathbb{C}=\text { the set of complex numbers }
\end{array}
$$
在Logic部分，作者提到了一种很有意思的证明方式（可能是我数学太渣了）。如果A和B是mathematical assertions，并且用-A和-B来表示他们的对立面（negations），那么A推出B意味着-B可以推出-A。

如果X和Y都是集合，
$$
X \times Y
$$
笛卡尔乘积（Cartesian product），是他们的有序对的集合。理解起来可能很拗口，一个直观的例子是假设集合A={a, b}，集合B={0, 1, 2}，则两个集合的笛卡尔积为{(a, 0), (a, 1), (a, 2), (b, 0), (b, 1), (b, 2)}。
